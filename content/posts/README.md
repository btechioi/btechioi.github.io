# 🔰 Blog post ->  
A blog post is like a conversation, you need to explain the information however you must only speak about subjects you understand and like. 
## 1️⃣ Intro 
* A promise statement 
* A preview of what's to come 
## 2️⃣ Overview 
* A simple definition 
* Examples 
## 📃 Steps 
* Detail of each the steps
